About the testquizquestions engine....

EMC-Prog's testquizquestions engine

Version 2.0

The testquizquestions engine is a simple and easy to modify JavaScript piece of code that lets anyone make their own "flashcards" without the need of propietary systems, as the data is stored permanently in the Javascript code itself, in the JSON format.

Changelog from V. 1.0: The UI is much more clear, and the code has been rewritten from the ground up to ensure that future versions can add new feaures, such as the soon-to-be error highlighting. Questions now won't be repeated until the whole session has been finished (the session can be restarted by reloading the webpage). Also, the Windows 7 UI has been used for a cleaner webpage (thanks to 7.css).
