# Testquizquestions

Un conjunto de programas web diseñados para aprender temas concretos de diferentes materias escolares.

Cuestionarios de aprendizaje disponibles:
* [Lengua castellana - Figuras retóricas](demo/figret/figuras.html)
* [Llengua catalana - Derivació de paraules](demo/catderivs)

Testquizquestions es un proyecto R.A.M.O.N.A. (Recursos Automatizados para la Memorización y Organización de Nociones Académicas), es decir, que está diseñado para ayudar a aprender conceptos de memoria, pero no puede asegurar un aprobado ni mucho menos un correcto aprendizaje de las materias.
